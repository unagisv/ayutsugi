# キャラ素材生成プロンプト集（案A・第1弾用）

| 項目 | 内容 |
|---|---|
| 対象 | specs/06 §5 の素材パイプライン第1弾（おとな男女） |
| 使い方 | 画像生成AI（画像編集・差分生成ができるもの推奨）にコピペして生成 → `web/dev/atelier.html` で取り込み |
| 規格 | 正面・直立・両腕は体側・全身・背景は単色（できれば無地のマゼンタ#FF00FFか白）・足元の影なし |

---

## 0. 共通スタイル指定（全プロンプトの先頭に付ける）

```
高精細ドット絵（ピクセルアート）、レトロゲームのキャラクタースプライト、
アニメ調で可愛らしい配色、やわらかい陰影、輪郭は濃い茶色、
全身立ち絵、真正面、直立、両腕は自然に体の横、表情は穏やか、
背景は無地一色、足元に影を描かない、1人のみ、文字を入れない
```

※お使いのツールでサンプル画像（歩いて育む！ドット人間ライフ）と同じ設定・モデルがあればそれを優先してください。スタイルの一貫性が最重要です。

## 1. ベースキャラ（最初の1枚・最重要）

以後の全素材はこの1枚の「編集（差分生成）」で作るため、気に入るまで作り直してください。

```
【共通スタイル指定】＋
20代の女性。標準体型。肩までのセミロングの茶色い髪（基準髪）。
ニットセーターとプリーツスカートの普段着（標準的な暮らしの服装）。
靴はショートブーツ。アクセサリなし。
```

→ 採用したら `base_adult_f.png` として保存。

男性ベースも同様に：

```
【共通スタイル指定】＋
20代の男性。標準体型。短い茶色の髪（基準髪）。
シャツとズボンの普段着（標準的な暮らしの服装）。スニーカー。アクセサリなし。
```

→ `base_adult_m.png`

## 2. 差分生成の鉄則

ベース画像を渡して「**○○だけ変えて、それ以外（ポーズ・体型・画風・配色・位置）は一切変えない**」と指示します。1回の編集で変えるのは1要素だけ。

### 2-1. 体型差分（×2）

```
この画像の人物の体型だけを「ほっそりと引き締まった体型」に変えてください。
顔・髪型・服のデザイン・ポーズ・画風は一切変えないでください。
```
```
この画像の人物の体型だけを「ふくよかな体型」に変えてください。
顔・髪型・服のデザイン・ポーズ・画風は一切変えないでください。
```

### 2-2. 服差分（各体型ベースに対して）

質素（heavy/normal/slim 各体型版で生成）：
```
この画像の人物の服だけを「質素な普段着（無地のTシャツと簡素なズボン、サンダル）」に
変えてください。体型・顔・髪型・ポーズ・画風は一切変えないでください。
```
立派：
```
この画像の人物の服だけを「上品で立派な装い（女性：きれいめのワンピースとブーツ／
男性：ジャケットとスラックスと革靴）」に変えてください。
体型・顔・髪型・ポーズ・画風は一切変えないでください。
```

### 2-3. 髪型差分（前髪・後ろ髪用。第1弾は6種）

髪は**必ず基準の茶色のまま**生成（色はゲーム側で変換します）。

女性向け：
```
この画像の人物の髪型だけを「高い位置のポニーテール」に変えてください。
髪の色は同じ茶色のまま。顔・服・体型・ポーズ・画風は一切変えないでください。
```
同様に：「ツインテール」「ロングストレート」「ボブ（おかっぱ）」
男性向け：「短く刈ったナチュラルショート」「ツンツンと逆立てた短髪」「センターパート」

### 2-4. 表情差分（×3）

```
この画像の人物の表情だけを「明るくにこにこした笑顔（頬に赤み）」に変えてください。
それ以外は一切変えないでください。
```
```
この画像の人物の表情だけを「元気がなくしょんぼりした顔（伏し目・への字口）」に
変えてください。それ以外は一切変えないでください。
```

### 2-5. アクセサリ差分（第1弾は3種）

```
この画像の人物に「赤いリボンの髪飾り」だけを追加してください。
それ以外は一切変えないでください。
```
同様に：「丸めがね」「小さなショルダーバッグ」

## 3. 第1弾の必要素材一覧（チェックリスト）

atelier で取り込むときの出力名。**太字はベースから直接**、他は差分。

| # | 生成物 | atelier出力名（adult_f の例） |
|---|---|---|
| 1 | **ベース女**（=標準体型・標準服） | body_casual_normal ほか分解出力 |
| 2 | 体型ほっそり | body_casual_slim |
| 3 | 体型ふくよか | body_casual_heavy |
| 4-9 | 服 質素/立派 × 3体型 | body_humble_{fit} / body_fine_{fit} |
| 10-12 | 表情 ふつう/にこにこ/しょんぼり | face_normal / face_happy / face_sad |
| 13-18 | 髪型6種（前髪/後ろ髪に分解） | hair_{style}_front / hair_{style}_back |
| 19-21 | 小物3種 | acc_ribbon / acc_glasses / acc_bag |

男性（adult_m）も同様。**まずは女性ベース1枚＋体型2差分だけ作って atelier に通し、品質と整合を確認してから量産**するのがおすすめです。

## 4. 品質チェックリスト（生成のたびに確認）

- [ ] 正面・直立・両腕は体側（ポーズがベースとズレていない）
- [ ] 全身が入っている（頭や足が切れていない）
- [ ] 背景が単色（透過 or 一色。複雑な背景はNG）
- [ ] 足元に影がない
- [ ] 差分生成で「変えていないはずの部分」が変わっていない（特に顔と輪郭）
- [ ] 髪は基準の茶色のまま
- [ ] 利用規約上、ゲームへの利用が許可されているツール・プランで生成している
